02 / 07

The O(n²)
Wall.

O(n²)
Attention cost grows quadratically. Double the context length, pay four times the compute.
Memory
KV cache grows linearly with n. At 100K tokens, storing past context alone consumes gigabytes per request.
1M tokens
Standard attention at 1M context requires roughly 1TB of GPU memory. It is not a software limit. It is physics.
Root cause
Every token attends to every other token. The Transformer was never designed for infinite sequence length.

03 / 07

The State Space
Model.

# Continuous-time formulation
h'(t) = A·h(t) + B·x(t)
y(t) = C·h(t) + D·x(t)

# h(t) = compressed state (all past context)
# A = state transition matrix
# B = input projection
# C = output projection
# O(n) time, O(1) memory per step

The state h(t) compresses the entire past into a fixed-size vector. No KV cache. No quadratic blowup. One matrix multiply per token.

04 / 07

Mamba:
Selective State.

# S4: fixed matrices (not input-aware)
A, B, C = learned_constants

# S6 (Mamba): input-dependent selection
dt, B, C = linear(x) # per token
A = -exp(A_log) # structured

# dt: how much to update state
# B: what to write into state
# C: what to read from state

Selective memory is the key insight. The model learns which tokens to remember and which to forget, per context. Standard SSMs apply the same filter to everything.

05 / 07

Linear Scaling
in Practice.

O(n)
Mamba scales linearly with sequence length. A 1M token sequence costs the same per-step as a 1K token one.
5× faster
Mamba inference throughput vs. Transformer at sequence length 2K. Gap widens as context grows.
Parallel scan
Hardware-efficient parallel scan replaces sequential recurrence during training. GPU-friendly by design.
Tradeoff
Fixed-size state compresses history. Precise recall of specific past tokens is harder than in attention-based models.

06 / 07

Jamba:
Best of Both.

Hybrid
Alternates Transformer attention and Mamba SSM layers. Not either/or. Uses each where it excels.
Why mix
Attention handles precise token recall. SSM layers provide long-range memory cheaply. Together they cover both failure modes.
256K ctx
Jamba-1.5 handles 256K token context with 3× less memory than a pure Transformer at the same scale.
AI21
Developed by AI21 Labs. 52B active parameters from a 398B total MoE model. Open weights available.

07 / 07

Quick
Reference.

SSM
State Space Model. Linear recurrence with structured matrices. O(n) time, O(1) memory per step.
S4
Structured SSM. First practical SSM for deep learning. Fixed transition matrices. Foundation for Mamba.
Mamba
Selective SSM (S6). Input-dependent A, B, C. Hardware-efficient scan. Linear scaling. No attention needed.
Jamba
Hybrid Transformer and Mamba. AI21 Labs. 256K context, 3× memory savings. Open weights.
Use when
Long sequences: genomics, audio, long documents, code. Wherever O(n²) attention is a hard constraint.
