02 / 07

The Vocabulary
Mismatch.

Teacher
GPT-4 has 100K+ vocabulary tokens. Each token maps to its own probability in the output distribution.
Student
Llama has 32K tokens. Different tokenizer. Different subword splits. Incompatible distribution shape.
Problem
KL divergence requires matching distributions. The standard KD approach silently breaks on mismatched vocabularies.

Standard knowledge distillation assumes teacher and student share the same vocabulary. Most real-world model pairs don't.

When Tokens
Don't Align.

// Same word, different tokenizations

Student: "unhappiness" → ["un","happiness"]

// Two failure modes in standard KD:
Unmatched → token in teacher only,
ignored entirely by student
Suppressed → teacher prob near zero,
forces student dist. wrong

Standard KD silently discards unmatched tokens. X-Token recovers them with a hybrid loss instead of throwing them away.

04 / 07

The Projection
Matrix.

Sparse
Non-zero only where tokens semantically overlap. Keeps the projection lightweight and interpretable.
Guided
Built from token alignment rules — prefix/suffix matching, subword containment, vocabulary intersection.
Static
Computed once before training starts. No extra cost per step — reused across the full training run.
Maps
Student token space → Teacher token space. Enables KL divergence across mismatched vocabularies.

05 / 07

Two Losses,
One Fix.

P-KL — Projected KL (matched tokens)
Project student dist. into teacher space
Compute KL on aligned token subsets
Handles vocabulary overlap correctly

H-KL — Hybrid KL (unmatched tokens)
Mix teacher dist. with uniform prior
Prevents zero-probability suppression
Recovers signal from unmatched tokens

H-KL is the key insight. Rather than ignoring unmatched tokens, it uses a soft target to avoid forcing the student to assign zero probability to tokens the teacher has never seen.

07 / 07

Quick
Reference.

Std KD
Standard Knowledge Distillation. Breaks when teacher and student vocabularies differ. Silently drops unmatched tokens.
X-Token
NVIDIA's framework. Sparse projection matrix + P-KL + H-KL. Works across any tokenizer pair.
P-KL
Projected KL divergence. Handles matched token subsets via sparse projection into teacher space.
H-KL
Hybrid KL divergence. Recovers suppressed and unmatched tokens using a soft uniform prior mixture.
Use
When distilling across tokenizers — e.g., GPT-4 teacher into Llama student for open-weight alignment.
