02 / 07

Why RSA
Feels Safe
Today.

The problem
RSA encryption is based on integer factorization. Given N = p x q where p and q are large primes, find p and q. Multiplying two 1,024-bit primes together takes microseconds. Factoring the result on classical hardware takes longer than the age of the universe.
Classical hardness
The best classical factoring algorithm is the General Number Field Sieve, running in sub-exponential time O(exp(n^(1/3))). A 2,048-bit RSA key would take a classical supercomputer roughly 300 trillion years to factor. Security comes entirely from this asymmetry.
Internet dependency
RSA and elliptic curve cryptography underpin TLS, SSH, PGP, code signing, and certificate authorities. Every HTTPS connection you make today is protected by the assumption that integer factorization is computationally infeasible. Shor's algorithm breaks that assumption.
Harvest now
Nation-state adversaries are harvesting encrypted traffic today to decrypt when quantum computers arrive. Long-lived secrets like government communications, medical records, and financial models need post-quantum protection now, not when quantum computers become available.

03 / 07

Qubits and
Superposition.

Classical bit
A classical bit is either 0 or 1. A register of 3 bits holds exactly one of eight possible states at any given moment. To search all 8 states you run 8 separate computations sequentially.
Qubit
A qubit exists in a superposition of 0 and 1 simultaneously until measured. A register of 3 qubits exists in all 8 states at once. Operations on that register apply to all 8 states in parallel. This is quantum parallelism.
Entanglement
Two entangled qubits share a correlated quantum state regardless of physical distance. Measuring one instantly determines the other. Shor's algorithm uses entanglement to build a quantum Fourier transform that extracts the period of a function exponentially faster than any classical approach.
Decoherence
Qubits are fragile. Any interaction with the environment collapses the superposition. Google's Sycamore processor operates at 15 millikelvin, colder than outer space, to maintain coherence. Current quantum computers have error rates of 0.1-1% per gate, limiting useful circuit depth.

Shor's
Algorithm.

# Shor's Algorithm to factor N: # Step 1: Classical - pick random a a = random_coprime(N) # gcd(a,N) = 1 # Step 2: Quantum - find period r of f(x) = a^x mod N # f(x) is periodic: f(x) = f(x+r) # QFT finds r in O(log N)^2 time r = quantum_period_finding(a, N) # Step 3: Classical - extract factors if r % 2 == 0: p = gcd(a**(r//2) - 1, N) q = gcd(a**(r//2) + 1, N) # Done: N = p * q, RSA is broken # Classical: O(exp(n^1/3)) -> centuries # Quantum: O((log N)^2) -> hours

05 / 07

Post-Quantum
Cryptography.

NIST PQC
In 2024 NIST finalized the first three post-quantum cryptography standards. ML-KEM (Kyber) for key encapsulation, ML-DSA (Dilithium) for digital signatures, and SLH-DSA (SPHINCS+) for hash-based signatures. All are based on mathematical problems believed hard for both classical and quantum computers.
Lattice problems
ML-KEM and ML-DSA are based on the hardness of the Learning With Errors problem over lattices. Given a lattice with small noise added to points, find the hidden structure. No efficient quantum algorithm for LWE is known. Key sizes are 2-4x larger than RSA equivalents.
Migration timeline
TLS 1.3 already supports hybrid classical/post-quantum key exchange. Chrome 124 ships X25519+Kyber768 by default. Google and Apple began migrating internal systems in 2022. NIST requires federal agencies to migrate to PQC by 2030.
Cryptographically relevant
A quantum computer capable of breaking RSA-2048 would need roughly 4,000 error-corrected logical qubits. Current machines have thousands of physical qubits but only tens of error-corrected logical qubits. Most estimates put a cryptographically relevant quantum computer 10-20 years away.

06 / 07

In
Production.

Google
Announced quantum supremacy in 2019 with 53-qubit Sycamore completing a sampling task in 200 seconds. 2024 Willow chip: 105 qubits with error correction improving as qubits scale. Below threshold for quantum error correction for the first time. Internal TLS already uses Kyber hybrid mode.
IBM
IBM Quantum roadmap targets 100,000 qubit systems by 2033. IBM Condor shipped 1,121 qubits in 2023. Focus on quantum error correction via the surface code. IBM Research partners include Boeing, ExxonMobil, and Cleveland Clinic for drug discovery and optimization.
Signal / iMessage
Signal Protocol added post-quantum key exchange via PQXDH in 2023. Apple iMessage added PQ3 using ML-KEM for forward secrecy in 2024. Both protect against harvest-now-decrypt-later attacks on long-term encrypted message archives.
AWS
AWS Certificate Manager supports hybrid TLS with ML-KEM. AWS Braket provides access to quantum processors from IonQ, Rigetti, and OQC. Amazon's own key management infrastructure began testing post-quantum algorithms internally in 2022.

Quick
Reference.

Shor's algorithm
Quantum algorithm for integer factorization running in O((log N)^2) time. Published by Peter Shor in 1994. Would factor RSA-2048 in hours given a large enough error-corrected quantum computer, versus centuries for classical algorithms.
Qubit
Quantum bit that exists in superposition of 0 and 1 until measured. N qubits represent 2^N states simultaneously. Decoherence from environmental interaction collapses superposition. Current error rates of 0.1-1% per gate limit useful circuit depth.
QFT
Quantum Fourier Transform: the key subroutine in Shor's algorithm. Finds the period of the modular exponentiation function exponentially faster than the classical FFT. This period reveals the prime factors of N.
ML-KEM
Module Lattice Key Encapsulation Mechanism, formerly Kyber. NIST's primary post-quantum key exchange standard, finalized 2024. Based on Learning With Errors hardness over lattices. Chrome 124 uses X25519+ML-KEM768 hybrid by default for all TLS connections.
Harvest-now
Nation-state strategy of recording encrypted traffic today to decrypt after quantum computers mature. Long-lived secrets need PQC protection now. NIST requires federal agencies to complete migration by 2030.
