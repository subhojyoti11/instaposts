▲ THE EVIDENCE

THE SOURCE CODE LEAK.

On March 31, 2026, Anthropic accidentally shipped their entire Claude Code source on npm. 512,000 lines. Researchers tore through it.

Found: 330+ utility files, 55 slash commands, 44 feature flags, ~200 environment variables.

NOT found: L99, /ghost, /godmode, BEASTMODE appear nowhere in the codebase. No hidden parser. No secret unlock system. Zero evidence.

The truth: Claude is trained on developer docs and technical writing. When you type "L99," it recognizes the intent for expert-depth response. The label isn't magic. The instruction clarity is.

THE TRUTH.

• Artifacts · Projects · Memory · MCP

• Extended thinking (auto in Claude Code)

• /effort · /init · /compact · /clear · /cost

• XML tags · Computer Use · 1M context

TRICKS (work via intent, not parsing):

• /godmode · BEASTMODE · /jailbreak
